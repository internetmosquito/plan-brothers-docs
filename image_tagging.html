process_images_amazon(folder_name=None)¶

Labels images specified in folder with Amazon Rekognition. Looks like Amazon does not support sending more then one image per request. This can be improved using Amazon Batch but is overkill for a small set of images like the one we are going to use :param folder_name: The complete path where the images are :param store_results: Indicates if obtained response should be stored as JSON file :return: A DataFrame containing the available data

process_images_clarifai(folder_name=None)¶

Processes the specified image folder using the Clarifai API :param folder_name: The complete path where the images are :return: A DataFrame containing the available data

process_images_google_vision(folder_name=None)¶

Iterates over the specified folder and returns the combined response from calling Google Cloud Vision API using only LABEL detection and 5 maximum per image. Since it looks like Google does not like sending more than 10 images per Request, we have to make sure we process every batch of 10 images until finished :param folder_name: The full path to the folder with images to be processed :return: A DataFrame containing the available data

process_images_microsoft_vision(folder_name=None, operation='tag')¶

Iterates over the specified folder and returns the combined response from calling Microsoft Vision API calling the tag endpoint just for now. Looks like Microsoft only allows one image per request, so we will have to deal with an image per request. :param folder_name: The full path to the folder with images to be processed :param operation: Specifies the API operation to use, current values allowed are tag and analysis :return: A DataFrame containing the available data

use_all(folder_name, pickle, store)¶

A wrapper that will use all available APIs :param folder_name: The folder containing images to be tagged :param pickle: If true, will try to load DataFrame from pickled file :param store: If true, will try to save DataFrame in pickled file :return: A DataFrame with all available data
